# Salud mental en el ámbito de la tecnología

## Objetivo

Este proyecto prepara y estudia la información recabada en el estudio de 2014 para poder localizar posibles relaciones entre la salud mental y los empleos relacionados con la tecnología.

## Proceso

Se ha procedido a limpiar, preparar y presentar la información obtenida de los participantes con Python para un análisis más claro de las tendencias. Con toda esa información se ha creado un dashboard en Power BI 
para ilustrar los datos.
